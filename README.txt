Crypto Trading App - Installation Guide

1️⃣ Extract the ZIP file.

2️⃣ Run the installer:

   📌 **Windows** → Double-click `install.bat`
   📌 **Linux/macOS** → Open terminal and run:
       chmod +x install.sh
       ./install.sh

3️⃣ Once installation is complete, **click the "Start Crypto GUI" icon** on your desktop.

4️⃣ The backend will continue running in the background, and the GUI can be restarted anytime.

5️⃣ If you want to stop everything, open a terminal and run:
       docker-compose down

Enjoy your trading bot! 🚀
